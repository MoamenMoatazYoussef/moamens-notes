	- Lifecycle Methods.
	- fetching.
	- Server-side search.
	- Client caching.
	- Error handling.
	- Jest snapshot tests.
	- Enzyme unit tests.
	- PropTypes.
	- 'ref' in a DOM element.
	- Using lodash.
	- setState but with callback functions.
	- Deploying.
